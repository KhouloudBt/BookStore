This project is a desktop application of a an online bookstore as part of an academic project
	
## Technologies
Project is created with:
* Java (jdk 1.8)
* JavaFx
* SceneBuilder
	
## dependecies
To run this project you will need to install the following libraries:
* JDBC MYSQL Driver
* ControlsFx
